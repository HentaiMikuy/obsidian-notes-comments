# Notes Comments

Obsidian 插件：在笔记中选中文字后添加勾划标记，并给标记绑定留言或 note 补充内容。

## 功能

- 选中文字后运行命令 `添加标记留言`，写入留言并生成标记。
- 同一个标记可以追加多条留言，留言会以对话线程的形式展示。
- 每条留言都支持独立编辑和删除；删除线程中的最后一条留言时，会同时移除正文标记。
- 标记样式可在插件设置中自定义。
- 内置预设：黄色荧光、蓝色批注、绿色虚线、玫瑰波浪。
- 留言展示方式可选：
  - 鼠标移动到标记处时，底部中间向上滑出留言框。
  - 鼠标移动到标记处时，在标记附近弹出留言框。
  - 鼠标移动到标记处时，中间右侧滑出留言框。
- 留言窗口会限制最大高度，长留言和多条留言会在内容区滚动，不会无限撑高弹窗。
- 留言列表会突出显示备注人名称，并弱化时间信息，便于区分作者和正文。
- 支持命令 `编辑光标处的标记留言` 和 `删除光标处的标记留言`。

## 使用方式

1. 在编辑器中选中需要标记的文字。
2. 运行命令 `添加标记留言`，或在右键菜单中选择 `添加标记留言`。
3. 输入备注人和留言内容，保存后正文中会生成标记。
4. 鼠标移动到标记处即可查看留言线程，并可继续追加、编辑或删除留言。

## 设置

- `留言展示方式`：可选择底部中间滑出、标记处弹出或中间右侧滑出。
- `默认备注人名称`：新增或编辑留言时，如果备注人名称留空，就使用这个名称。
- `默认标记样式`：新增标记留言时默认使用的样式。
- `样式预设`：可以新增、删除、重置或调整标记样式，包括背景色、文字色、下划线颜色和下划线样式。

## 开发

```bash
npm install
npm run build
```

构建后会生成 `main.js`。作为 Obsidian 插件使用时，将 `manifest.json`、`main.js` 和 `styles.css` 放在 vault 的 `.obsidian/plugins/obsidian-notes-comments/` 目录下并启用插件。

## 数据格式

插件会在 Markdown 中写入类似下面的 span，以便标记能随笔记内容保存：

```html
<span class="onc-comment-mark" data-onc-id="..." data-onc-style="yellow-marker">被标记的文字</span>
```

其中 `data-onc-id` 是标记线程 ID。同一标记处的多条留言会共享这个线程 ID，留言内容、样式和展示方式保存在插件的 `data.json` 中。
